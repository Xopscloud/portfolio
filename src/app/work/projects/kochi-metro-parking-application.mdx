## Project Overview

Designed and deployed in GCP with a fully containerized (Docker) environment, implementing load balancer, NAT, internet gateway, Cloud Armor, and GitLab CI/CD pipelines.

## Technologies

- GCP
- Docker
- GitLab CI/CD
- Cloud Armor
- Load Balancer

## Key Features

- Containerized deployments with Docker
- Automated pipelines with GitLab CI/CD
- Network design including NAT and internet gateway
- DDoS protection and WAF via Cloud Armor
- Load-balanced, highly available architecture
