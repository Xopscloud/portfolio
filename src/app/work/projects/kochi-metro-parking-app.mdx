## Project Overview

Designed and deployed the Kochi Metro Parking Application in Google Cloud Platform (GCP) with a fully containerized Docker environment. The project implemented comprehensive cloud infrastructure including load balancing, security, and automated CI/CD pipelines.

## Architecture & Infrastructure

### GCP Services Implemented
- **Containerization**: Docker containers for application deployment
- **Load Balancing**: Google Cloud Load Balancer for traffic distribution
- **Network Security**: Cloud Armor for DDoS protection and WAF
- **Networking**: NAT Gateway and Internet Gateway configuration
- **CI/CD**: GitLab CI/CD pipelines for automated deployment

### Security Implementation
- Cloud Armor security policies
- Network access controls
- Automated security scanning
- SSL/TLS encryption

## Technologies Used

- **Cloud Platform**: Google Cloud Platform (GCP)
- **Containerization**: Docker
- **CI/CD**: GitLab CI/CD
- **Security**: Cloud Armor, NAT Gateway
- **Load Balancing**: GCP Load Balancer
- **Infrastructure**: Terraform (IaC)

## Key Achievements

- Fully automated deployment pipeline
- High availability and scalability
- Enhanced security with Cloud Armor
- Cost-optimized infrastructure
- Comprehensive monitoring and logging
